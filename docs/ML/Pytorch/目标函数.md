# 目标函数

## 交叉熵 CrossEntropyLoss

$$
\begin{align}
H(p, q) = -\sum_{x\in \chi}{p(x)\ log\ q(x)}
\end{align}
$$

!!! tip
    \(\chi\) 表示样本空间（Sample Space），也就是所有可能出现的样本或事件的集合。在这个公式 \( H(p, q) = -\sum_{x\in \chi}{p(x)\ \log\ q(x)} \) 中，\(\chi\) 是一个集合，包含了所有可能的 \( x \) 值。

    \(\chi\) 就是所有可能事件的集合，可以把它看作一个“全集”。

用于训练**分类模型**
